Kebijakan Privasi & Keamanan
🛡️

Terakhir diperbarui: 11 Mei 2026

1. Data yang Kami Kumpulkan

Kami menghormati privasi Anda. Satu-satunya data yang kami kumpulkan adalah:

  • ID Pengunjung: ID acak yang dibuat otomatis oleh browser Anda untuk membedakan pengguna (tanpa informasi pribadi).
  • Preferensi & Interaksi: Model AI yang sering Anda gunakan dan kategori pertanyaan (contoh: Koding, Gambar) untuk memberikan rekomendasi yang lebih baik.

2. Data yang TIDAK Kami Kumpulkan

  • Riwayat Obrolan: Semua percakapan Anda disimpan 100% di perangkat lokal (Browser) menggunakan LocalStorage. Kami tidak pernah mengirim atau menyimpan teks obrolan Anda ke server database kami.
  • Data Pribadi: Kami tidak meminta nama, email, nomor telepon, atau alamat IP yang dilacak secara persisten.

3. Keamanan Data

Kami menerapkan protokol keamanan berlapis:

  • Enkripsi HTTPS: Semua data yang dikirim antara browser dan server dienkripsi menggunakan SSL/TLS.
  • Supabase RLS: Database menggunakan Row Level Security untuk mencegah akses tidak sah.
  • Rate Limiting: Sistem kami memblokir permintaan berlebihan (spam/DDoS) untuk menjaga kestabilan dan keamanan layanan.
  • Sanitasi Input: Semua teks yang masuk dibersihkan dari kode berbahaya (XSS) sebelum diproses atau ditampilkan.

4. Hak Anda

Anda memiliki kendali penuh atas data Anda. Anda dapat:

  • Menghapus riwayat obrolan lokal kapan saja melalui menu Riwayat.
  • Merubah atau menghapus preferensi yang tersimpan melalui menu Pengaturan.
